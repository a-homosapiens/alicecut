Quick Start

AliceCut Help

Create caption-led videos from lyrics, subtitles, audio, and video.
Import captions

Choose File → Import Lyrics and open an LRC, SRT, VTT, or timestamped text file. If captions already exist, choose whether to replace the primary captions or add the file as another caption track.

Add video and audio

Use File → Import Video or Import Audio. Imported media appears as clips on the timeline. Unsupported video formats are converted automatically for editing.

Arrange the timeline

Drag clips and captions to move them. Drag either edge to trim timing. Select captions to edit text, timing, position, and per-line effects.

Style and preview

Use the style panels to choose aspect ratio, typography, colors, background, and entrance or exit effects. Press Space to preview from the playhead.

Save your project

Choose File → Save Project to create an AliceCut project file before making larger edits or closing the application. Keep the project file and its media folders together when copying the project to another computer; AliceCut stores portable relative paths but does not embed the media itself.

Export

Choose File → Export Video, select format and quality settings, then choose an output location. You can also export the selected caption track as SRT.
